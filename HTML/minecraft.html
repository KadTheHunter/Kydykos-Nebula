Minecraft

Leaders, Citizens and Automatons of the Kædykos Nebula all play on the TotalFreedom Minecraft server.

You can connect to the Hub via play.totalfreedom.me or directly to the Freedom Server via freedom.play.totalfreedom.me.

For coordinates to the Kædykos Nebula, or it's outposts and property, please contact KaddicusAlmighty in-game via the Chat if he's online, or via the /mail command if he's offline.

Be sure to check out the TotalFreedom Forums as well, for the latest news about TotalFreedom.
