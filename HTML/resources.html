Resources

Below is a list of resources commonly used in the Kædykos Nebula

MCStacker.net
- MCStacker is one of the best resources for creating Custom Items and altering NBT. If you're new to Item Creation and NBT, MCS is a great place to start.

Minecraft Armor-Stand
- Minecraft-ArmorStand allows you to create custom armor stands, with a visual model of the stand itself to help you.

NCS Banner
- NCS Banner lets you design banner patterns, and generates the chosen command for obtaining them in-game.

Colorize.fun
- Colorize.fun helps in colorizing your Minecraft text with solid colors and gradients

Minecraft Chat Symbols
- Minecraft Chat Symbols has a list of symbols, emoticons and characters supported by Minecraft and useable in chat and on signs.

MapartCraft
- MapartCraft lets you upload an image, and converts it into a Minecraft Map. You can customize Blocks, Dithering, Staircasing, etc, and then download either the resulting image, a Schematic file, or a map data file.

R74n
- R74n is an interesting amalgamation of resources and tools for Minecraft and other things. Of note, is the Box Page containing the /give commands and NBT for a number of Shulker Kits, including Ryan UwU 1-7.

Kad's Current Mods
- Kad's Current Mods is a list of all Mods currently in use on Sir Kaddicus III's main Minecraft instance.
